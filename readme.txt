
это краткое описание, как работать с программой(в самом внизу есть пример)

1)запустить скрипт RamblerSql.py, он переведёт весь ваш dataset в объекты базы данных, появится файл с этой базой данной. 
При запуске предварительно надо указать путь в PARAM.xml на папку с dataset. К примеру, 
<PathOfData>C:\\Users\\Alex\\Desktop\\courseWorkFromCloud\\DataMining</PathOfData> 
в вкладке PathOfData. 
2)Запустить создатель графа СreateGraph.py, в файле PARAM.xml, также надо указать какие месяца будем анализировать. 
к примеру так
<Tables>info_cli_otgr_201502,info_cli_otgr_201503,info_cli_otgr_201504</Tables>
3)Искать сами правила SearchRules.py, для этого надо выбрать параметр CHRG, что он значит будет в презентации. 
Его также указываем в PARAM.xml и выбираем на каком(орьентированном или нет) графе будет анализ, в вашей версии рекомендую оставлять 1(Орьентированный) isDirected
<isDirected>1</isDirected> 
<CHRG>0.5</CHRG>
//картинка для понимания прилагается 
шаг 1) делается всего один раз, после его повторять не надо для данного набора данных
шаг 2) делается для каждого нового поднабора данных, которые хотите исследовать 
шаг 3) выполняется для одно поднабора данных с разными параметрами CHRG 
0)Если ничего не работает! во-первых, писать мне(ALEX.ALEX.95@MAIL.RU), во-вторых, можно 
открыть в ide, проверить, что все библиотеки подключены. 
Лучше запускать в PyCharm тогда всё будет уже в External Libries. Попробовать писать pip для нужных файлов.
Код открыт, если что можно попробовать что-то самому подкорректировать. 

структура файла с параметрами:
<PARAM>
    <Tables>info_cli_otgr_201502,info_cli_otgr_201503,info_cli_otgr_201504</Tables>
    <isDirected>1</isDirected>
    <CHRG>0.5</CHRG>
    <PathOfData>C:\\Users\\Alex\\Desktop\\courseWorkFromCloud\\DataMining</PathOfData>
</PARAM>

Пример сценария действий: 
известно: 
файлы лежат в папке C:\\Users\\Alex\\Data
хотим исследовать два месяца info_cli_otgr_201505,info_cli_otgr_201506
хотим запустить для коэфицента CHRG=0.7
0)заполняем PARAM.xml этими параметрами
<PARAM>
    <Tables>info_cli_otgr_201505,info_cli_otgr_201506</Tables>
    <isDirected>1</isDirected>
    <CHRG>0.7</CHRG>
    <PathOfData>C:\\Users\\Alex\\Data</PathOfData>
</PARAM>
1)открываем консоль(можно и в ide, лучше в ide, рекоменндую pycharm)
2)пишем в консоле RamblerSql.py (ждём выполнения, построится база данных)
3)пишем в консоле CreateGraph.py (ждём выполнения, построится граф)
4)пишем в консоле SearchRules.py (находит правила, если менять значения в PARAM.xml можно получать новые выборки, не надо выполнять шаги 2) и 3), 
сразу поменяли и запускаем 4)) смотрим либо вывод, либо береём результаты из файла rules.txt, там они тоже сохранятся.
